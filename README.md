# Baby Name Generator

En enkel webbapplikation för att generera babynamn.

## Beskrivning

Applikationen visar ett slumpmässigt babynamn i mitten av skärmen. Klicka på knappen för att generera ett nytt namn. Listan innehåller 72 populära namn (36 pojknamn och 36 flicknamn).

## Användning

Öppna `index.html` i en webbläsare.

## Funktioner

- Stort, lättläst namn centrerat på skärmen
- Knapp för att generera nytt namn
- Filtrera namn efter kön (alla, pojke, flicka)
- Spara favoritnamn i en lista
- Favoriter sparas i webbläsaren (localStorage)
- Ta bort namn från favoriter
- Mjuk övergångsanimation vid namnbyte
- Responsiv design för mobil, surfplatta och desktop

## Filer

- `index.html` - Huvudsida med HTML-struktur
- `styles.css` - Styling och responsiv layout
- `app.js` - JavaScript för namngenering, filtrering och favoriter
